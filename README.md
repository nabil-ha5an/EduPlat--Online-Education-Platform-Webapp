# EduPlat--Online-Education-Platform-Webapp
A webapp made using Laravel and PHP as an academic project that enables students to subscribe to subject-specific courses on a monthly basis while providing administrators with full CRUD (Create, Read, Update, Delete) capabilities.

### Project Demonstration
[![Watch the Demo](https://img.youtube.com/vi/9mAgjj95UF8/maxresdefault.jpg)](https://youtu.be/9mAgjj95UF8)
*Click the image above to watch the EduPlat walkthrough.*

### Technical Stack:
Laravel, PHP, HTML, and CSS

### Key Features: 
* Adheres to the MVC structure
* Students can add courses to their carts from various subjects
* Students can select the number of months of subscription
* Admins can add, update, delete, and set the availability of new subjects and courses
* Users can add a profile picture and update their bio
